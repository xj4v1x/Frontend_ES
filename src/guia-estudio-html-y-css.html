Guía de estudio HTML y CSS

Esta documentación está diseñada para ayudar a principiantes a aprender HTML y CSS desde cero.

Introducción a HTML y CSS

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son los dos pilares fundamentales para la creación de páginas web.

Mientras que HTML se encarga de la estructura y el contenido de la página, CSS define su apariencia y diseño.

Comprender cómo funcionan juntos es esencial para el desarrollo de proyectos web efectivos y atractivos.

Estructura básica de un documento HTML

Cada documento HTML se organiza en una estructura específica con la declaración <!DOCTYPE html> en la parte superior, seguida de las etiquetas <html>, <head> y <body>.

Dentro de <head>, se definen metadatos como el título (<title>) y enlaces a hojas de estilo o scripts.

El elemento <body>, contiene todo el contenido visual de la página, como texto, imágenes, enlaces,etc...

HTML

<!DOCTYPE html>

<html>lang = "es"

<head>

<title>Mi Primera Página Web</title>

</head>

<body>

<h1>Hola, Mundo!</h1>

<p>Este es un párrafo de ejemplo.</p>

</body>

</html>

¿Qué es CSS?

CSS es el lenguaje de estilos utilizado para describir la presentación de un documento HTML.

Permite definir colores, fuentes, espaciados y posicionamiento de los elementos en una página, mejorando la experiencia visual y la usabilidad.

Cómo CSS Interactúa con HTML

CSS se integra con HTML de tres maneras principales:

  • Estilos en línea. Se aplican directamente en la etiqueta HTML mediante el atributo style.
  • Estilos internos. Se incluyen en la sección <head> del HTML utilizando la etiqueta <style>.
  • Estilos externos. Se guardan en un archivo separado .css y se enlazan al HTML con la etiqueta <link>.

Cada método tiene sus usos específicos, pero el uso de hojas de estilo externas es el más recomendado por la facilidad de mantenimiento y reutilización.
